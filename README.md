# Challenge Lampp-it

Aplicação de cadastro e listagem de clientes desenvolvido com NextJS, Chakra UI, React Hook Form, Zod, JSON Server, Axios.

## Instruções

Instalar as dependências 

```bash
npm install
```

Iniciar a aplicação 

```bash
npm run dev
```

Iniciar o server

```bash
npm run server
```
